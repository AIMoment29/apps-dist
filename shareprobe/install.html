安装 ShareProbe
  1. 1用 Safari 或 Chrome 打开本页（微信里点右上角「···」→「在浏览器中打开」。若点安装没反应，请改用 Safari）
  2. 2点上面的 安装，弹窗里再点「安装」，回桌面等它装好
  3. 3先点桌面的 App 图标试试能不能打开 —— 能打开就完成了，下面不用看
  4. 4打不开？按提示对号入座：
    · 提示"不受信任" → 设置 → 通用 → VPN与设备管理 → 信任证书
    · 提示"需要开发者模式" → 设置 → 隐私与安全性 → 最底部 开发者模式 → 打开 → 重启并确认
    · 提示"无法验证 / Unable to Verify" → 关掉 VPN 或换用蜂窝数据，再点图标重试

请在 iPhone 上打开本页安装（推荐 Safari）。
若提示"无法安装"，多为本机未在开发者账号登记，请把设备 UDID 发给开发者登记后重试。
